### 平时自己新建项目时 总需要配置es6、ts、less、react|vue|node相关生态的一些配置，整理一个cli工具一键生成模板

# 快速开始

1. 安装   
`npm i -g quick-demo`  
`npm i -g quick-demo --registry=https://registry.npm.taobao.org` 

2. 运行
`quick-demo`  
> 会在当前目录下创建你输入的project name

### 支持创建的项目

- normal
   - 主要支持js开发的demo
   - 支持typescript
- react  
   - 主要支持react开发的demo
   - react@^16.9.0  typescript
- vue（未完待续）
   - 主要支持vue开发的demo
- node（未完待续）
   - 主要支持node开发的demo
- lib（未完待续）
   - 主要支持js-library开发的demo
